## 🚀  BRAND NEW - E-Commerce Project (Evolution)
Este repositorio documenta mi proceso de aprendizaje y desarrollo como desarrollador frontend, pasando desde la concepción de una identidad de marca hasta la implementación de interfaces interactivas complejas.

## 📈  Mi Proceso de Aprendizaje
A lo largo de este proyecto, he evolucionado de crear maquetas estáticas a desarrollar una experiencia de usuario (UX) coherente y dinámica.

## 1. Concepto e Identidad Visual
Contraste de Estilos: Desarrollé dos lenguajes visuales distintos para la misma marca: una estética Cyber-grunge/Metálica para el área de seguridad (Login) y una estética Pop Art/Editorial con colores primarios para la bienvenida (Welcome).

Consistencia de Marca: Implementé una identidad sólida utilizando tipografías audaces (Bebas Neue) y elementos gráficos recurrentes para unificar la experiencia.

## 2. Dominio de CSS e Interactividad (UI/UX)
Efectos de Revelado (Hover): Aprendí a manejar propiedades avanzadas de CSS como opacity, filter: blur(), y transform para crear transiciones donde el contenido se revela de forma elegante al interactuar con el mouse.

Micro-interacciones: Implementé botones con efecto inverso (hover de negro/blanco a blanco/negro) para proporcionar feedback visual inmediato y mejorar la navegabilidad.

Solución de Problemas Técnicos: Logré resolver conflictos de interactividad relacionados con la especificidad de CSS y el bloqueo de eventos mediante la propiedad pointer-events.

## 3. Arquitectura y Lógica con Vue.js
Navegación SPA: Implementé Vue Router para conectar diferentes vistas (Welcome -> Login) sin recargar la página, asegurando una experiencia fluida.

Estructura de Componentes: Aprendí a organizar el código en componentes de Vue, separando la lógica del template y los estilos encapsulados (scoped).

## 🛠️ Tecnologías Utilizadas
Vue.js 3 (Composition API)

Vite (Entorno de desarrollo)

CSS3 (Flexbox, Grid, Animaciones y Transiciones)

Vue Router (Gestión de rutas)

## 📂 Vistas del Proyecto
Welcome Page: Una explosión de color (Amarillo, Rojo, Azul) que segmenta al usuario por categorías (Men, Girls, Couples) con efectos de desenfoque interactivos.

Login Page: Una interfaz minimalista y técnica con texturas metálicas y elementos Y2K.

Catálogo: Estructura de cuadrícula (Grid) limpia para la visualización de productos, enfocada en la facilidad de escaneo para el usuario.
